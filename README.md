## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/eric4479/spray-foam-website.git
   cd spray-foam-website
   ```

2. **Serve locally**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```
